# Weather Dashboard

An interactive weather dashboard that displays real-time weather data, a 5-day forecast, and detailed weather metrics for cities around the world.

Built with JavaScript, HTML, CSS, OpenWeatherMap API

---

## Features

- Current weather conditions with temperature, humidity, wind, and UV index
- 5-day forecast with high and low temperatures
- City search with recent search history
- Toggle between Fahrenheit and Celsius
- Sunrise and sunset times
- Dynamic weather icons that change based on conditions
- Fully responsive for mobile and desktop

---

## How to Use

1. Open index.html in your browser
2. Search any city in the search bar
3. Press Enter or click Search
4. Toggle between F and C using the unit buttons

## Supported Cities

New York, Los Angeles, Chicago, Miami, London, Tokyo, Paris, College Park and more

---

## Tech Stack

- Frontend: HTML, CSS, Vanilla JavaScript
- API: OpenWeatherMap API
- Deployment: GitHub Pages
